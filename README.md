## Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/khalidshams-tech/flask-learning-labs.git
```

2. Open the project folder:

```bash
cd flask-learning-labs
```

3. Review the project folders and follow any instructions inside each lab folder.

If a lab contains a Flask app, the usual setup is:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install flask
python app.py
```

On macOS/Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install flask
python3 app.py
```
